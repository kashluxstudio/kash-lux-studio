# Ka$h Lux Studio Website

A responsive static business website prepared for GitHub Pages.

## Files

- `index.html` — website structure and copy
- `styles.css` — layout, colors, and responsive styling
- `script.js` — mobile navigation and automatic copyright year

## Before publishing

1. Open `index.html`.
2. Find `YOUR-EMAIL-HERE`.
3. Replace it with the email that should receive consultation requests.
4. Update prices, social links, project examples, and legal-page links as needed.

## Publish with GitHub Pages

1. Create a GitHub repository.
2. Upload these three website files to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`.
6. Save.

Do not place passwords, payment credentials, private customer data, or API secrets in this public repository.
